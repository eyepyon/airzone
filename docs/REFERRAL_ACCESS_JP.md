# 紹介コード・SNSシェア機能へのアクセス方法

## 🎯 アクセス方法

### 方法1: ダッシュボードから（推奨）

1. **ログイン**
   - Googleアカウントでログイン

2. **ダッシュボードに移動**
   - `/dashboard` にアクセス

3. **Referralボタンをクリック**
   - Quick Actionsセクションの「🎁 Referral」ボタン

### 方法2: 直接アクセス

ブラウザのアドレスバーに入力：
```
https://airz.one/referral
```

## 📱 紹介ページの機能

### 1. 紹介コードカード

```
┌─────────────────────────────────────┐
│ 友達を招待                          │
│                                     │
│ あなたの紹介コード                  │
│ ┌──────────┐                        │
│ │ ABC12345 │ [コピー]               │
│ └──────────┘                        │
│                                     │
│ 紹介リンク                          │
│ https://airz.one/signup?ref=ABC...  │
│                                     │
│ SNSでシェア:                        │
│ [Twitter] [LINE] [Facebook]         │
└─────────────────────────────────────┘
```

### 2. 統計情報

- **総紹介数**: 招待した友達の総数
- **成功**: 登録完了した友達の数
- **クリック数**: リンクがクリックされた回数
- **獲得コイン**: 紹介で獲得したコインの合計

### 3. 最近の紹介

- 紹介した友達のリスト
- ステータス（完了/保留中）
- 獲得したコイン数
- 登録日時

## 🎯 よくある質問

### Q1: 紹介コードはどこで確認できますか？

**A**: `/referral` ページの一番上のカードに表示されています。

### Q2: 紹介リンクが機能しているか確認できますか？

**A**: はい、統計情報の「クリック数」で確認できます。

### Q3: コインはいつ付与されますか？

**A**: 友達が登録を完了すると自動的に付与されます（通常は即座）。

### Q4: 紹介できる人数に制限はありますか？

**A**: いいえ、制限はありません。何人でも紹介できます。

### Q5: 自分で自分の紹介コードを使えますか？

**A**: いいえ、自己紹介は無効です。

### Q6: 紹介コードを変更できますか？

**A**: 現在は変更できません。自動生成された8文字のコードが割り当てられます。

### Q7: SNSボタンが動作しません

**A**: 
1. ポップアップブロッカーを無効化
2. ブラウザの設定を確認
3. 別のブラウザで試す

## 🛠️ トラブルシューティング

### エラー: "統計の取得に失敗しました"

**解決策**:
1. ページを再読み込み（F5）
2. ログアウト→ログイン
3. ブラウザのキャッシュをクリア

### 紹介コードが表示されない

**解決策**:
1. ログインしているか確認
2. ページを再読み込み
3. ブラウザのコンソール（F12）でエラーを確認

### SNSシェアボタンが開かない

**解決策**:
1. ポップアップブロッカーを確認
2. ブラウザの設定でポップアップを許可
3. 手動でリンクをコピー＆ペースト
