# Escrowステーキング機能へのアクセス方法

## 🎯 アクセス方法

### 方法1: ダッシュボードから（推奨）

1. **ログイン**
   - Googleアカウントでログイン

2. **ダッシュボードに移動**
   - トップページから「Dashboard」をクリック
   - または直接 `/dashboard` にアクセス

3. **Stakingボタンをクリック**
   - Quick Actionsセクションの「💎 Staking」ボタン

### 方法2: 直接アクセス

ブラウザのアドレスバーに入力：
```
https://airz.one/staking
```

### 方法3: ナビゲーションメニュー（実装予定）

トップバーのメニューから「Staking」を選択

## 📱 ステーキングページの使い方

### タブ1: キャンペーン一覧

**表示内容**:
- 現在開催中のステーキングキャンペーン
- 各キャンペーンの詳細情報
- 報酬NFTのプレビュー

**操作**:
1. 参加したいキャンペーンを選択
2. 「ステーキングする」ボタンをクリック
3. ステークするXRP量を入力
4. 確認して送信

### タブ2: マイステーク

**表示内容**:
- 自分が参加中のステーキング
- 過去のステーキング履歴
- ステータス（アクティブ/完了）

**情報**:
- ステーク量
- 解放予定日
- キャンペーン名
- 現在のステータス

## 🎯 よくある質問

### Q1: ステーキングページが見つかりません

**A**: 以下を確認してください：
1. ログインしているか
2. URLが正しいか (`/staking`)
3. ページを再読み込み（F5）

### Q2: キャンペーンが表示されません

**A**: 現在開催中のキャンペーンがない可能性があります。管理者が新しいキャンペーンを作成するまでお待ちください。

### Q3: ステーキングボタンが押せません

**A**: 以下の理由が考えられます：
- キャンペーンが終了している
- 定員に達している
- 最小ロック量を満たしていない

### Q4: ステーク中にキャンセルできますか？

**A**: いいえ、EscrowでロックされたXRPは期間終了まで引き出せません。これがブロックチェーンの安全性を保証しています。

### Q5: NFTはいつ受け取れますか？

**A**: ロック期間終了後、自動的にウォレットに送信されます。`/nfts` ページで確認できます。

## 🛠️ トラブルシューティング

### エラー: "データの取得に失敗しました"

**解決策**:
1. インターネット接続を確認
2. ページを再読み込み
3. ログアウト→ログイン

### エラー: "ステーキングに失敗しました"

**解決策**:
1. ウォレットに十分なXRPがあるか確認
2. 最小ロック量を満たしているか確認
3. キャンペーンがアクティブか確認

### ページが真っ白

**解決策**:
1. ブラウザのキャッシュをクリア
2. 別のブラウザで試す
3. ブラウザのコンソール（F12）でエラーを確認

## 📊 管理者向け: キャンペーン作成

管理者は以下の方法でキャンペーンを作成できます：

### 方法1: 管理パネル

```
管理パネル → Escrow Campaigns → 新規作成
```

### 方法2: API

```bash
POST /api/v1/admin/escrow/campaigns
{
  "name": "XRPステーキングキャンペーン",
  "description": "XRPをロックして限定NFTを獲得",
  "min_amount_drops": 100000000,
  "lock_days": 30,
  "nft_reward_name": "Airzone Staker NFT",
  "max_participants": 1000
}
```
